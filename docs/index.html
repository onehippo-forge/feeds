Introduction

What is the Feed Creator?

Feed Creator was previously known as the RSS Feed Creator Project

Feed Creator is an addon for Hippo CMS that lets you create RSS feeds, ATOM feeds and provides an API which enables you to create your own web syndications with the CMS. You can create as many feeds as you like, and for each feed you can setup filters to determine which documents to include.

Code base on GitHub

The code of this project is on GitHub at https://github.com/onehippo-forge/feeds/.
